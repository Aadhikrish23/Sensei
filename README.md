# Sensei – AI Interview Training Platform

Sensei is an AI-powered system that helps developers prepare for job interviews.

## Features

- Resume upload and parsing
- Job description management
- Resume ↔ JD skill matching
- AI interview sessions
- Readiness scoring

## Tech Stack

Frontend
- React
- Tailwind
- MUI

Backend
- Node.js
- Express
- Prisma
- PostgreSQL

AI Service
- FastAPI
- OpenAI API

Database
- PostgreSQL
- MongoDB
